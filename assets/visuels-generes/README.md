# assets/visuels-generes/ — référence de TON

Ce dossier se remplit au fil de l'eau avec tes **images IA brutes** (sorties de `image-linkedin`, copiées depuis `outputs/images/` quand une génération te plaît).

Il sert de **référence de ton** : humour, niveau de second degré, type de références culturelles détournées, ambiance. Quand tu génères une nouvelle image, l'agent s'en inspire (sans copier) pour rester cohérent avec ce qui te ressemble.

Vide au départ. Tu le nourris en y déposant tes meilleures générations, avec un nom parlant (ex. `moi-en-detective-vintage.jpg`).
